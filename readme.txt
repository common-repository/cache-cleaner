== Installation ==

1. Go to Plugins Page
1. Add New
1. Search for "Cache Cleaner"
1. Install & Activate
1. Navigate to Cache Cleaner in your admin menu
1. Choose How Often you Need the Plugin to Clean your Cache. 
1. Choose the Plugins you are Using and you Need to Activate the Scheduled Clean Up.
1. Ready to go !
